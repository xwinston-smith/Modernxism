# ปัญหา และ ที่มาของ Modernxism
ประเทศไทยเป็นเป็นประเทศที่ใช้ระบบรวมศูนย์ ใครก็ตามที่สามารถยึดอำนาจศูนย์กลางได้ ก็สามารถควบคุมทุกอย่างในประเทศได้ ประเทศไทยยังเป็นหนึ่งในประเทศที่มีการรัฐประหารเป็นอันดับต้นๆของโลก ครั้งล่าสุดเมื่อปี 2557 กบฏที่เข้ามายึดอำนาจนั้น เป็นเพียงกลุ่มคนเพียงหยิบมือ แต่สามารถควบคุมทั้งประเทศได้อย่างง่ายดาย สร้างระบบเพื่อกอบโกยผลประโยชน์ให้กับตนเอง และพวกพ้อง สร้างความเสียหายแก่ประเทศมหาสาร ณ เวลานั้น ประชาชนไม่สามารถทำอะไรได้เลย การจะออกมาเรียกร้องก็อาจจะนำมาซึ่งการนองเลือด นั่นจึงเป็นสาเหตุที่ทำให้ข้าพเจ้าครุ่นคิดอยู่เสมอว่า จะมีวิธีไหนไหมที่ทำให้ประเทศนี้เป็นของประชาชนจริงๆ เป็นประเทศที่ประชาชนมีอำนาจในการกำหนดชะตาชีวิตของตัวเขาเอง

เป้าหมายของการยึดอำนาจนั้นชัดเจนว่า กบฏเหล่านั้นทำไปเพื่อต้องการอำนาจ แต่ในโลกทุนนิยมที่เงินนั้นสำคัญกว่าสิ่งใด อำนาจที่กบฏเหล่านั้นต้องการ ล้วนมีเป้าหมายปลายทางคือเงิน หากปราศจากเงิน หากอำนาจที่ได้มานั้นไม่สามารถนำมาซึ่งเงินทองได้ ก็ไม่มีเหตุจูงใจใดในการทำเรื่องดังกล่าว

ระบบการเงินที่เป็นอยู่ในปัจจุบัน เงินที่เราใช้อยู่ในทุกวันถูกควบคุมด้วยธนาคารแห่งชาติ และถูกควบคุมโดยรัฐบาลกลาง นั่นหมายความว่า ถ้าใครก็ตามสามารถยึดอำนาจศูนย์กลางได้ ก็จะสามารถควบคุมระบบการเงินได้เช่นกัน จะเป็นอย่างไรหากระบบการเงินแทนที่จะเป็นของศูนย์ธนาคารกลาง แต่เป็นของประชาชน จะเป็นอย่างไรที่ระบบการเงิน มีคนทุกคนในประเทศเป็นเจ้าของและมีอำนาจในการควบคุมโดยสมบูรณ์ หากประชาชนสามารถควบคุมระบบการเงินได้ แม้แต่กบฏที่ยึดสภาได้ แต่หากอำนาจที่ได้มานั้นไม่สามารถนำมาซึ่งเงิน อำนาจนั้นก็ไร้ซึ่งความหมาย และไม่มีเหตุผลอีกต่อไปที่จะทำการกบฏ ประเทศจะเป็นของประชาชนอย่างแท้จริง

ประชาชนสามารถเป็นเจ้าของระบบการเงินได้โดยใช้เทคโนโลยี [Blockchain](https://en.wikipedia.org/wiki/Blockchain)

# Blockchain คืออะไร?
<img src="../../imgs/Database, a computer for storing data.png" alt="Database, a computer for storing data" width="750px">

**_พระราชาจะดำรงอยู่ได้ เพราะความเชื่อของประชาชน หากประชาชนหมดศรัทธา ชายผู้นั้นก็ไม่ใช่ราชาอีกต่อไป_** เงินก็เช่นกัน ลองจินตนาการว่านาย ก. เป็นคนไม่เอาถ่าน นาย ก. ไม่ทำงาน นาย ก.ไม่ทำอะไรที่เป็นประโยชน์ต่อสังคม นาย ก. รังแต่จะสร้างความวุ่นวายให้แก่สังคม วันหนึ่ง นาย ก. เดินไปเจอเงินของนาย ข. ที่ฝังไว้ นาย ก. ร่ำรวยเพียงข้ามคืนจากการโขมย คำถามคือ นาย ก. ควรค่าแก่เงินก้อนนั้นหรือไม่

หากเงินเก็บไว้ที่ปัจเจค ถึงแม้ว่านาย ก. จะไม่ได้สร้างมูลค่าใดๆเลยให้แก่สังคม แต่ถ้าหากเงินอยู่ในมือนาย ก. มันก็จะเป็นของนาย ก. โดยสมบูรณ์

**Blockchain** คืออะไร ลองจินตนาการว่า นาย A, B, และ C เป็นคนในสังคม นาย A, B, และ C มีเงินคนละ 100 บาท แต่แทนที่จะสร้างเงินขึ้นมาและให้แต่ละคนถือไว้ เงินจะถูกเก็บเป็นข้อมูลแทน และข้อมูลเหล่านั้นจะถูกเก็บไว้ที่ทุกคน (ตัวอย่างภาพ B1.0) โดยทุกคนที่ถือสมุดนั้นจะรู้ว่าใครมีเงินเท่าไหร่ ลองจินตนาการว่า A, B, และ C มีสมุดคนละ 1 เล่ม โดยสมุดนี้จะเก็บข้อมูลจำนวนเงินของคนทุกคนเอาไว้ ตัวอย่างดังภาพ B2.0

<img src="../../imgs/Blockchain Village Transaction Analogy Figure B2.0.png" alt="Blockchain Village Transaction Analogy Figure B2.0" width="750px">

ถ้าหากว่า นาย A อยากจะซื้อปลาจากนาย B, นาย A และ นาย B ก็จะสร้างรายการทางการเงิน ในสมุดของตนเอง และหลังจากที่ทั้งคู่สร้างรายการของแต่ละฝั่งแล้ว ทั้งคู่ก็จะบอกกับคนที่เหลือให้เก็บข้อมูลเดียวกันเพื่อให้ข้อมูลเงินในบัญชีของคนทุกคนที่เหลือได้รับการอัพเดต ดังภาพ B3.0

<img src="../../imgs/Blockchain Village Transaction Analogy Figure B3.0.png" alt="Blockchain Village Transaction Analogy Figure B3.0" width="750px">

หลังจากที่นาย A และ นาย B กระจายข้อมูลให้กับคนที่เหลือครบทุกคนแล้ว ตอนนี้เงินในบัญชีของทุกคนก็อัพเดตเป็นที่เรียบร้อย ตามภาพ B4.0 จะเห็นได้ว่าเงินไม่ได้ถูกเก็บที่ตัวบุคคล แต่ถูกเก็บไว้ที่คน **_ทุกคน_** ที่ใช้ระบบเดียวกัน นี่เป็นตัวอย่างคร่าวๆของระบบ [Blockchain](https://en.wikipedia.org/wiki/Blockchain)

# ค่าเงิน และ เศรษฐกิจ

<img src="../../imgs/Modernxism Village Network Analogy Figure B5.0.png" alt="Modernxism Village Network Analogy Figure B5.0" width="750px">

ตอนนี้เรามีคน 3 คนในหมู่บ้านนั่นคือ นาย A, B , และ C จะเป็นอย่างไรถ้ามีการรับสมาชิกเพิ่ม หรือ มีคนที่ต้องการเข้ามาใช้ระบบเงินเดียวกัน อย่างแรก เงินในระบบจะมีอยู่อย่างจำกัด นั่นหมายความว่าถ้าหากมีคนที่ต้องการใช้เงินเพิ่มแต่เงินมีจำกัด คนที่มีเงินอยู่เดิมทั้งหมดจะต้องตกลงกันว่าจะแบ่งเงินกันอย่างไร ยกตัวอย่างเช่น ตอนนี้ นาย A, B, และ C มีเงิน 90, 110, และ 100 ตามลำดับ และมีคนที่ต้องการมาใช้ระบบเงินเดียวกันอีก 3 คนนั่นคือ นาย X, Y, และ Z การที่นาย A, B, และ C จะยอมให้นาย X, Y, และ Z มาใช้ระบบเดียวกัน และแบ่งเงินกัน นาย X, Y, และ Z จะต้องนำบางอย่างที่มีมูลค่าที่สมเหตุสมผลมาแลก สมมติว่า นาย X, Y, และ Z มีข้าวคนละ 1 กระสอบ และต้องการแลกเป็นเงิน กระสอบละ 50 บาท สมมติว่า นาย A, B, และ C ยินยอม, นาย A, B, และ C ก็จะให้สมุดการเงินหมู่บ้านให้กับสมาชิกใหม่ทั้งสาม พร้อมเงินที่แลกกับข้าว 3 กระสอบ ตอนนี้สมุดของคนทุกคน ทั้ง A, B, และ C และ X, Y, และ Z ก็จะเป็นดังภาพ B6.0

<img src="../../imgs/Modernxism Village Network Analogy Figure B6.0.png" alt="Modernxism Village Network Analogy Figure B6.0" width="750px">

จะเห็นได้ว่าเงินในระบบจะยังคงมีเท่าเดิม แต่สิ่งที่เปลี่ยนไปคือ มูลค่าของเงิน ก่อนหน้านี้เรามีสมาชิก 3 คน แต่ตอนนี้เรามีเพิ่มมาเท่าตัว นั่นหมายความว่า ค่าเงินก็จะแข็งขึ้นหรือมีมูลค่าเพิ่มขึ้นเท่าตัวเช่นกัน ยกตัวอย่างเช่น วันแรก นาย A ซื้อ ปลาจาก นาย B เป็น เงิน 10 บาท พอวันต่อมานาย A อยากจะขายปลาให้นาย X นาย A จะไม่สามารถขายในราคาเท่าวันแรกได้เพราะค่าเงินมีมูลค่ามากขึ้น นาย A ต้องขายปลาในราคา 5 บาท พอฟังมาถึงจุดนี้แล้วเหมือนว่านาย A จะขาดทุนเพราะ ต้องขายราคาถูกกว่าที่ซื้อมาตอนต้น แต่หากลองมาคิดดีๆแล้ว ถ้านาย A ขายปลาได้ 5 บาทจริงๆ เงินที่ได้นั้น ก็สามารถซื้อปลาได้ในจำนวนเท่าเดิม กล่าวคือ กำลังการซื้อของนาย A จะยังคงเท่าเดิมถึงแม้จำนวนเงินที่ได้มาจะน้อยกว่าก็ตาม

<img src="../../imgs/Blockchain Village Transaction Analogy Figure B7.0.png" alt="Blockchain Village Transaction Analogy Figure B7.0" width="750px">

จะเป็นอย่างไรถ้าหากเราไม่นำเงินไปแลกเป็นสินค้าก่อนที่ค่าเงินจะมีมูลค่าเพิ่มขึ้น คำตอบคือ คนที่ถือเงินไว้ก็จะมีมูลค่าเพิ่มขึ้น กล่าวคือ ได้กำไร ส่วนคนที่นำเงินไปแลกเป็นสินค้าก่อน ก็จะยังคงมีกำลังการซื้อเท่าเดิม

หากสมาชิกยังคงเพิ่มขึ้น เหตุการณ์ดังกล่าวก็จะเกิดขึ้นซึ่งไม่มีผลกระทบในเชิงลบต่อสมาชิกผู้ถือเงิน พอถึงจุดหนึ่งที่ไม่มีสมาชิกเพิ่มขึ้นแล้ว หรือเปลี่ยนแปลงในเปอร์เซ็นที่น้อยมากเช่น ถ้าหากมีสมาชิก 70 ล้านคน ระบบก็จะหมุนเวียนไปเรื่อยๆตามกลไกตลาดในที่สุด

# Citizen Nodes Network

ถ้าหากว่าประเทศไทยใช้ระบบ Modernxism ประเทศเราจะหน้าตาเป็นอย่างไร สมมติว่าตอนนี้เรามี 70 ล้านคน อย่างแรกเลยเราก็ต้องมีบางอย่างที่เก็บรายชื่อของคนทั้งประเทศก่อน นั่นก็คือ Citizen Nodes Network มันก็คือ เครื่อข่าย blockchain ที่เก็บรายชื่อของคนในประเทศ โดยที่คนทุกคนในประเทศ จะมีคอมพิวเตอร์ คนละ 1 เครื่อง (หรือ ครอบครัวละเครื่อง) คอยเก็บข้อมูลเหล่านี้ไว้ โดย เครือข่ายนี้จะมีความสำคัญเพราะ คนที่จะมีสิทธิในการโวทนโยบายต่างๆ หรือ มีสิทธิในการถือเงินจะต้องเป็นคนใน citizen network เท่านั้น

<img src="../../imgs/Database, a computer for storing data.png" alt="Database, a computer for storing data" width="750px">
<img src="../../imgs/Modernxism Citizen Network C2.0.png" alt="Modernxism Citizen Network C2.0" width="750px">

ตอนนี้เรามี Blockchain ที่เก็บรายชื่อของคนทั้งประเทศแล้ว ทุกๆอย่างที่เกิดขึ้นในประเทศจะต้องผ่านการโวทของเสียงข้างมาก ไม่ว่าจะเป็นกฎหมาย นโยบาย และ อื่นๆ

แต่การจะโวทผ่านประชาชนหลักสิบล้านคนในทุกๆเรื่องเป็นเรื่องที่แทบจะเป็นไปไม่ได้ Modernxism เลยต้องการองค์กรณ์อิสระที่คอยทำหน้าที่นี้ คอยตัดสินใจในเรื่องต่างๆให้ และจ่ายผลตอบแทนเป็นภาษีตามรายได้ เพราะ Modernxism มีระบบการเงินที่โปร่งใส ทุกการใช้เงิน หรือ ทรัพย์สินที่มีทั้งหมดจะถูกเก็บข้อมูลไว้เพื่อใช้ในการคำนวณการเก็บภาษี

<img src="../../imgs/Modernxism Foreign Exchange.png" alt="Modernxism Foreign Exchange" width="750px">

คนที่ไม่ได้เป็นสมาชิกของ Citizen network จะไม่สามารถถือเงินได้โดยตรง แต่จะสามารถนำทรัพสินมาค้ำประกันไว้กับธนาคาร และธนาคารจะออกบัญชีที่สามารถใช้ในการทำธุรกรรมในนามของทางธนาคาร

หากคนในประเทศต้องการถือเงินตราต่างประเทศ ก็สามารถทำได้โดยการ นำเงินไปค้ำประกันไว้กับธนาคาร และรับเงินตราที่ธนาคารนั้นมี

# องค์กรตัวแทน

<img src="../../imgs/Representative Organisation.png" alt="Representative Organisation" width="750px">

องค์กรณ์ตัวแทนคือองค์กรที่มีประชาชนในประเทศเป็นคนดำเนินการ ตั้งแต่ 1 คนเป็นต้นไป ทำหน้าที่คอยตัดสินใจเรื่องต่างๆ และคอยดูแลสิทธิให้กับสมาชิก โดยที่สมาชิกจะให้ **_token_** ที่ใช้ในการยืนยันตัวตน ที่สามารถยกเลิกได้ตลอดเวลา หากองค์กรที่เลือกไม่สามารถตัดสินใจได้ตามเป้าประสงค์ก็สามารถ ยกเลิกสิทธิตัวแทน และเลือกองค์กรณ์ใหม่ที่เหมาะสมกับตนเอง

# การถ่วงดุลอำนาจผ่านระบบการเงิน

ประเทศไทยมีการทำรัฐประหารเป็นอันดับต้นๆของโลก และทุกครั้งประชาชนไม่สามารถหยุดยั้งได้ด้วยสันติวิธี ความคิดของการทำรัฐประหารมักจะเกิดจากกลุ่มคนที่มีอำนาจเพียงไม่กี่คน ในการลงมือปฏิบัติจริง คนเหล่านี้ต้องพึ่งพากำลังพล หรือ ทหารชั้นผู้น้อยเป็นผู้ดำเนินการ ทหารเหล่านั้นก็ยังต้องพึ่งพาโครงสร้าง และ คนในสังคมในการรับบริการต่างๆไม่ว่าจะเป็น น้ำ อาหาร พลังงาน และ อื่นๆ หากเราสามารถตัดการเข้าถึงสิ่งสำคัญเหล่านี้ได้ กลุ่มกบฏเหล่านี้ก็จะไม่สามารถมีชีวิตได้อีกต่อไปในสังคม หากต้องการเข้าถึงสิ่งต่างๆจากสังคม คนเหล่านี้ต้องทำในสิ่งที่เป็นประโยชน์แก่คนในสังคมเท่านั้นจึงจะได้รับอนุญาตให้เข้าถึงสาธารณูปโภคของคนในสังคม

สมมติว่าตอนนี้เราใช้ระบบ Modernxism ที่ประชาชนเป็นเจ้าของระบบการเงิน เงินของเราถูกเก็บไว้ที่ทุกคน และจำเป็นต้องถูกยอมรับจากทุกคน หากมีใครสักคนทำอะไรที่เป็นอันตรายกับคนในระบบหรือประเทศ คนอื่นๆในระบบสามารถ หยุดการเข้าถึงทรัพย์สินได้ชั่วคราว เพราะเงินถูกเก็บเป็นข้อมูลไว้ที่ทุกคน การจะเข้าถึงเงินได้ต้องได้รับการยินยอมจากคนในสังคม

ในกรณีของการทำรัฐประหาร หรือการสลายการชุมนุม หากมีทหารชั้นสูงสั่งให้ทหารชั้นผู้น้อยกระทำการเป็นอันตรายต่อประชาชน ประชาชนสามารถแบนการเข้าถึงทรัพย์สิน ประชาชนมีสิทธิหยุดการเข้าถึงทรัพย์สินของคนที่เกี่ยวข้องทั้งหมด หากเป็นกรณีที่ทหารเข้าทำรัฐประหาร ประชาชนมีสิทธิในการหยุดการเข้าถึงทรัพย์ทั้งหมดของคนทั้งกองทัพ ไม่ว่าจะรู้เห็นหรือไม่เกี่ยว คนทั้งกองทัพจะถูกระงับการเข้าถึงทรัพย์สิน เพื่อให้คนที่อยู่ในกองทัพที่ไม่เกี่ยวข้อง เกิดการถ่วงดุลอำนาจกันเอง ถึงแม้ว่าทหารผู้น้อยไม่มีส่วนรู้เห็นในการวางแผน แค่ทำตามผู้บังคับบัญชา ณ สถานที่จริงหากทหารชั้นผู้น้อยเหล่านั้นเลือกที่จะหันปืนเข้าหาประชาชน คนเหล่านี้ก็จะถูกแบนไม่ให้เข้าถึงเงิน ซึ่งหมายถึง อาหาร น้ำ ที่อยู่อาศัย รวมถึงครอบครัวของผู้กระทำการด้วย เพราะรายชื่อและข้อมูลทุกอย่างถูกเก็บไว้ที่ทุกคน หากกบฏเหล่านั้นต้องการมีชีวิตต่อไปในสังคม คนเหล่านี้จำเป็นต้องปกป้องประชาชน และถ่วงดุลอำนาจกับกบฏที่เหลือ
